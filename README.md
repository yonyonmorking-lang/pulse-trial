# Rhythm Game

Rhythm Game is a browser rhythm-memory game. Players listen to five short wooden tap patterns and recreate each rhythm through the microphone.

## Run locally

```bash
python3 server.py
```

Then open:

```text
http://127.0.0.1:4174
```

GitHub Pages can host the game UI as a static site. The Python leaderboard API is local-only unless a separate backend is deployed.
